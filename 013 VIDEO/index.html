HTML ENTITIES

HTML entities are used to display reserved characters in HTML. For example, the less than symbol (<) is a reserved character in HTML, so it must be represented as < to be displayed correctly.

Here are some common HTML entities:

  • & - Ampersand (&)
  • < - Less than (<)
  • > - Greater than (>)
  • " - Double quote (")
  • ' - Single quote (')
  •   - Non-breaking space
  • © - Copyright symbol (©)
  • ® - Registered trademark symbol (®)
